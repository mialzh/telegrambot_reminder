## Reminder bot - асинхронный Telegram-бот для напоминаний

Бот позволялет создавать, просматривать и удалять напоминания с привязкой к конкретному времени.  
Напоминания отправляются автоматически в заданный момент(с точностью до минуты).

### Стек технологий
- **Python 3.13**
- **aiogram 3**
- **PostrgeSQL 15**
- **SQLAlchemy 2.0**
- **asyncpg**
- **Docker**

### Возможности
Регистриция пользователей происходит при первом применении команды `\start`.
- **Добавление напоминания с датой** `/add DDDD-MM-DD HH:MM текст_напоминания`
- Просмотр активных напоминаний `/list`. Вернется спосок, каждая строчка вида `ID | YYYY:MM:DD HH:MM текст`
- Удаление напомианния по ID `/remove ID`
- Фоновый планировщик **scheduler.py** каждые 59 секунд провеоряет, не просрочилось ли какое напоминение и отправляет, когда пришло время.
  При это фоновый планировщик не отправляет одно напоминание более одного раза.

### Запуск проекта
Проект можно запустить локально у себя на компьютере. Для этого достаточно скопировать репозиторий, изменить **.env.example** -> **.env** и в соответствующее место в этом файле ввести свой токен.
